hexo+github搭建个人博客：Hexo配置+NEXT主题配置

发表于 2019-08-19 | 分类于 博客

什么是HEXO

Hexo是一款基于Node.js的静态博客框架，依赖少易于安装使用，可以方便的生成静态网页托管在GitHub和Heroku上，是搭建博客的首选框架。这里我们选用的是GitHub
这是我的个人博客效果Byron‘s
以及个人github项目地址https://github.com/dingpc000/dingpc000.github.io

Github创建个人仓库

登录到GitHub，注册个人账号以后,新建repositories，并以username.github.io的形式命名，例如我的就是

安装Node.js

Hexo基于Node.js，点击链接下载安装包，注意安装Node.js会包含环境变量及npm的安装，安装后，检测Node.js是否安装成功，在命令行中输入，
node -V
v10.16.3
检测npm是否安装成功，在命令行中输入:
npm -v
6.9.0
到这里hexo的环境就已经配置完成

Hexo安装

Hexo就是我们的个人博客网站的框架， 这里需要自己在电脑常里创建一个文件夹，可以命名为Blog，Hexo框架与以后你自己发布的网页都在这个文件夹中。创建好后，进入文件夹中，按住shift键，右击鼠标运行Git Bash,使用npm命令安装Hexo，输入：
npm install -g hexo-cli
完成安装后，同样在该文件夹下，输入命令来初始化博客：
hexo init blog
为了检验我们已经完成配置，输入一下几个命令：

完成后打开浏览器输入地址：localhost:4000,可以看到博客显示
以下是常见的hexo命令：
npm install hexo -g #安装Hexo
npm update hexo -g #升级
hexo init #初始化博客

命令简写
hexo n “我的博客” == hexo new “我的博客” #新建文章
hexo g == hexo generate #生成
hexo s == hexo server #启动服务预览
hexo d == hexo deploy #部署

hexo server #Hexo会监视文件变动并自动更新，无须重启服务器
hexo server -s #静态模式
hexo server -p 5000 #更改端口
hexo server -i 192.168.1.1 #自定义 IP
hexo clean #清除缓存，若是网页正常情况下可以忽略这条命令

刚刚的三个命令依次是新建一篇博客文章、生成网页、在本地预览的操作。
